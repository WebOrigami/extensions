You can use Handlebars to generate a list of links to all files in the `pages` folder:

```console
$ ori links.hbs pages
<ul>
    <li>
      <a href="Alice.html">Alice.html</a>
    </li>
    <li>
      <a href="Bob.html">Bob.html</a>
    </li>
    <li>
      <a href="Carol.html">Carol.html</a>
    </li>
</ul>
```
